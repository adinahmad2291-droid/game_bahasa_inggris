# Generator Prompt Game Bahasa Inggris Anak

Project ini digunakan untuk membuat prompt otomatis untuk AI agar dapat membuat game edukasi bahasa Inggris anak.

## Fitur
- Generate prompt otomatis
- Mobile friendly
- Bisa deploy di GitHub Pages
- Copy prompt sekali klik
- Modern UI

## Cara Upload ke GitHub
1. Upload semua file ke repository GitHub
2. Aktifkan GitHub Pages
3. Selesai

## Struktur
- index.html
- script.js
